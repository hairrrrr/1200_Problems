给定一个长度为N的数列，求数值严格单调递增的子序列的长度最长是多少。

#### 输入格式

第一行包含整数N。

第二行包含N个整数，表示完整序列。

#### 输出格式

输出一个整数，表示最大长度。

#### 数据范围

1≤N≤1000，
−109≤数列中的数≤109

#### 输入样例：

```
7
3 1 2 1 8 5 6
```

#### 输出样例：

```
4
```

```cpp
#include<iostream>
#include<cstring>
#include<cstdio>
#include<algorithm>

using namespace std;

const int N = 1010;

int n;
int a[N], dp[N];

int main(void)
{
    scanf("%d", &n);
    
    for(int i = 1; i <= n; ++i) scanf("%d", &a[i]);
    
    for(int i = 1; i <= n; ++i)
    {
        // 自己本身组成一个最长子序列
        dp[i] = 1;
        for(int j = 1; j < i; ++j)
            if(a[j] < a[i]) dp[i] = max(dp[i], dp[j] + 1);
    }
    
    int res = 0;
    
    for(int i = 1; i <= n; ++i) res = max(res, dp[i]);
    
    printf("%d\n", res);
    
    return 0;
}
```
