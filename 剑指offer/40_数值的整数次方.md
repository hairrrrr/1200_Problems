实现函数double Power(double base, int exponent)，求base的exponent次方。不得使用库函数，同时不需要考虑大数问题。

 ```
示例 1:

输入: 2.00000, 10
输出: 1024.00000

示例 2:

输入: 2.10000, 3
输出: 9.26100

示例 3:

输入: 2.00000, -2
输出: 0.25000
解释: 2-2 = 1/22 = 1/4 = 0.25
 ```




说明:

-100.0 < x < 100.0
n 是 32 位有符号整数，其数值范围是 [−231, 231 − 1] 。

```cpp
class Solution {
public:
    double myPow(double x, int n) 
    {
        long long N = n;
        int flg = N < 0 ? 1 : 0;
        if(flg) N *= -1;
        double ans = 1;
        
        while(N)
        {
            if(N & 1) ans *= x;
            x *= x;
            N >>= 1;
        }

        if(flg)
            return 1 / ans;
        else 
            return ans;
    }
};
```
