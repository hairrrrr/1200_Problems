在字符串 s 中找出第一个只出现一次的字符。如果没有，返回一个单空格。 s 只包含小写字母。

```
示例:

s = "abaccdeff"
返回 "b"

s = "" 
返回 " "
```


限制：

0 <= s 的长度 <= 50000

```cpp
class Solution {
public:
    char firstUniqChar(string s) 
    {
        int letter[26] = {0};
        for(auto& ch : s)
        {
            ++letter[ch - 'a'];
        }
        for(int i = 0; i < s.length(); ++i)
        {
            if(letter[s[i] - 'a'] == 1)
                return s[i];
        }
        return ' ';
    }
};
```
